# Content Multiplication

Turn 1 piece of content into 15+ assets.

```
ONE IDEA → MULTIPLE FORMATS → ALL CHANNELS
```

## Standard Multiplication Matrix

From 1 long-form video or pillar post:

| Asset Type | Count | Platform |
|------------|-------|----------|
| Long-form video | 1 | YouTube |
| Shorts / clips | 3-5 | YouTube Shorts |
| Reels | 3-5 | Instagram |
| Carousel | 1-2 | Instagram |
| Stories | 5-10 | Instagram |
| TikToks | 2-3 | TikTok |
| Thread | 1 | Twitter/X |
| LinkedIn post | 1 | LinkedIn |
| Blog post | 1-2 | Website |
| Newsletter block | 1 | Email |
| Lead magnet section | 1 | Download |

**Total: 15-25 assets from 1 source**

## Multiplication Plan Format

```json
{
  "source": "YouTube: [Title]",
  "multiplication": {
    "shorts": ["Hook clip", "Key insight", "Tutorial clip", "Result reveal"],
    "reels": ["Full story (60s)", "Results only (30s)", "Tutorial (45s)"],
    "carousel": "5-slide breakdown of key framework",
    "thread": "10-tweet thread with evidence",
    "linkedin": "Professional angle with ROI metrics",
    "blog": "Expanded case study with step-by-step",
    "newsletter": "Behind-the-scenes personal angle"
  }
}
```

## Clip Selection Rules (for Shorts/Reels)

Good clip candidates:
- Strong hook moment (first 3 seconds of the long-form)
- Surprising claim or counterintuitive insight
- Clear before/after demonstration
- Step-by-step tutorial that stands alone
- Emotional high point or relatable confession

Bad clip candidates:
- Mid-thought transitions between topics
- Context-dependent sections that need the full video
- Sections with references to "earlier in the video"

## Platform Adaptation Requirements

| Platform | Required adaptation |
|----------|-------------------|
| YouTube Shorts | ≤60s, vertical 9:16, hook in first 2s |
| Instagram Reels | ≤90s, vertical, caption ≤150 words |
| TikTok | ≤60s for best reach, text overlays |
| Twitter Thread | 6-10 tweets, first tweet = standalone hook |
| LinkedIn | 150-300 words, professional framing, metrics |
| Newsletter | Personal angle, conversational, one CTA |

## Done Criteria

- [ ] Source content has transcript or key moments documented
- [ ] Multiplication plan created (all applicable formats)
- [ ] Clip list identified with timestamps
- [ ] Each platform format explicitly adapted
- [ ] Schedule plan includes all derivatives
