# Content Talent

Support on-camera recording sessions. Prepare everything before, assist during, and process after.

## Handoff Pack (Required Before Recording)

A complete handoff pack contains:

1. **Final script** — reviewed, camera directions included
2. **Storyboard** — shot-by-shot with visual descriptions
3. **Recording brief** — setup, props, slides needed
4. **B-roll list** — what supplementary footage to capture
5. **Teleprompter-ready text** — large font, natural pacing, no jargon

## Script Format for Talent

```
[SECTION: Hook — 0-3 seconds]
[LOOK AT CAMERA]
"[Opening line — bold, specific]"
[TEXT OVERLAY: "key stat or phrase"]

[SECTION: Problem — 3-10 seconds]
[SHOW SCREEN]
"[Problem description with specific detail]"

[SECTION: Solution — 10-45 seconds]
[ALTERNATE: camera + screen]
"[Step-by-step or narrative]"
[B-ROLL: screen recording of tool]

[SECTION: CTA — last 3 seconds]
[LOOK AT CAMERA]
"[Direct ask — follow, comment, click link]"
```

## Teleprompter Guidelines

- Large font size for arm's-length reading
- Natural pacing — write like you speak, not like you write
- Mark pauses with `[pause]`
- Mark emphasis with CAPS
- Keep lines short — one idea per line

## Post-Recording Processing

After recording session:
1. **Transcription** — generate transcript from audio
2. **Caption file** — SRT format for accessibility and text overlays
3. **Cut list** — note timestamps of usable takes
4. **B-roll slots** — mark where b-roll goes in the edit
5. **Thumbnail frames** — identify best freeze-frames

## Recording Session Rules

- No personal data visible on screen
- Clean desktop and relevant tabs only
- Test audio before rolling
- Capture 3-second silence before and after each take
- Record at least 2 takes for every section

## Done Criteria

- [ ] Final script reviewed and camera-direction annotated
- [ ] Storyboard attached
- [ ] Teleprompter-ready version prepared
- [ ] B-roll list created before session
- [ ] Post-recording: transcript + caption file + cut list
