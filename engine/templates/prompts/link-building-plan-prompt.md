# Prompt: Generate Link Building Plan

> Use with: Ship Engine engine · Stage: Awareness / SEO

---

## System Context

You are an SEO and growth strategist helping an early-stage founder build their first link building strategy.

---

## Prompt

```
Help me create a link building plan for my product.

**Context:**
- Product: {{product_name}}
- Website: {{website_url}}
- Current Domain Rating / Authority: {{current_da}} (0 if new)
- Target audience / ICP: {{icp_description}}
- Primary keywords to rank for: {{target_keywords}}
- Content I already have: {{existing_content}}
- Budget for paid placements: {{budget}} (USD/month, 0 = none)
- Time I can dedicate: {{hours_per_week}} hours/week

**Deliverables:**
1. Top 10 directories to submit to (with DA, cost, and category)
2. 5 guest post site targets (with topic ideas and outreach angle)
3. 3 partnership link opportunities (type + approach)
4. 1 "link bait" content idea tailored to my niche
5. A month-by-month action plan for the first 3 months
6. KPIs to track and how to measure them

Focus on tactics a solo founder can execute without an agency.
Format as a structured plan matching the link-building-plan.md template.
```

---

## Variables to Replace

| Variable | Description | Example |
|----------|-------------|---------|
| `{{product_name}}` | Product name | "LaunchKit" |
| `{{website_url}}` | Your domain | "launchkit.io" |
| `{{current_da}}` | Domain Rating (Ahrefs) or DA (Moz) | "12" |
| `{{icp_description}}` | Who buys your product | "solo SaaS founders, 1–5 person teams" |
| `{{target_keywords}}` | 3–5 keywords | "saas launch checklist, gtm for founders" |
| `{{existing_content}}` | Content you have | "5 blog posts, 1 guide" |
| `{{budget}}` | Monthly budget | "0" or "200" |
| `{{hours_per_week}}` | Time available | "3" |
